# 🧠 Echo

**Echo** is a cross platform AI chatbot built with **Flutter**, powered by **Google Gemini** and **Firebase Authentication**. Designed with a clean architecture approach, Echo delivers intelligent conversations while encouraging user engagement through a built in gamification system.

## ✨ Features

### 🤖 AI Chat
- Powered by Google Gemini
- Real-time conversational responses
- Clean chat interface
- Typing indicator

### 🔐 Authentication
- Firebase Email & Password Authentication
- Google Sign-In
- Persistent login sessions

### 🎯 Personalized Onboarding
- Interest selection
- User preference storage
- First-time onboarding flow

### 🏆 Gamification
- XP reward system
- Daily login streaks
- User levels
- Message counter
- Achievement-ready architecture

### 👤 Profile
- User information
- XP & Level overview
- Total messages sent
- Saved interests
- Logout functionality

# 🎮 Gamification System

Users earn XP by interacting with Echo.

| Action | XP |
|----------|---:|
| Daily Login | +10 |
| Complete Onboarding | +30 |
| Send Prompt | +5 |
| Receive AI Response | +2 |

Level progression increases automatically as XP grows.
